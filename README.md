
# RAG on SEC 10-K

This project implements a Retrieval-Augmented Generation (RAG) system
on Apple and Tesla 10-K filings.

## How to Run
1. Open the Colab notebook
2. Clone the repo(mentioned in colab)
3. Install requirements(requiremet.txt)
4. Upload PDFs
5. Call answer_question()
